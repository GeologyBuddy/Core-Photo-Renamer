# ⚠️ Windows SmartScreen Notice

Thank you for using this software.

The executable (`.exe`) file is located inside the `dist` folder.  
When launching the application for the first time, Windows may display a **SmartScreen warning**.

> This warning is common for independently developed applications that do not yet have a commercial code-signing certificate.

The software is safe to run, and you can proceed using one of the methods below.

---

# ✅ Method 1 — Run from the SmartScreen Warning Window

If the warning window appears:

1. Click **More info**
2. Click **Run anyway**

📷 Please refer to the included screenshots for guidance.

---

# ✅ Method 2 — Unblock the File from Properties

If Windows continues blocking the application, you can manually unblock it:

### Steps

1. Right-click the `.exe` file
2. Select **Properties**
3. Under the **General** tab, scroll to the bottom
4. In the **Security** section:
   - Enable the **Unblock** checkbox
5. Click **Apply**
6. Click **OK**

---

# 💡 Why Does This Happen?

Windows SmartScreen may flag newly distributed applications because:

- The application is newly published
- It is not yet digitally signed
- Windows has not built enough reputation data for the file

This is a standard Windows security feature and does not necessarily indicate that the software is unsafe.
